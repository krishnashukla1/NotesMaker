# Notes Maker App

A simple and effective MERN stack notes application to create, edit, and delete notes.

## Features

- CRUD for notes
- Markdown support
- MongoDB for storage
- Responsive React interface

## Installation

- Frontend: `npm install && npm start` inside `client`
- Backend: `npm install && npm start` inside `server`

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
